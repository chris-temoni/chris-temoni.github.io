Classes vs Ids

What is Classes?

In HTML, classes are attriutes which can be assigned to elements to group them together depending on the purpose. it is used to apply different styles or target certain elements with the use of CSS

Step 1: HTML

Give your "p" element the class "paragraph2"

Step 2: CSS

Navigate to your css file and add a class.

NOTE: when adding a class we would add the "." symbol along with the class name, along with the style of the paragraph e.g., width of the paragraph, font alignment of the text, margin of the text, the text size, and the color of the text, example indicated in the image below:

Result

As you can see, the output of the result has been defined by the class we created, with the text color being white, the text alignment being centered and the margin of 20px auto limit, and the text being 25px in size. NOTE: the blue background is a part of the body image and the outline border is a part of the actual image and not the class we created.

What is Id?

Quiet similar to classes but a selector is used mainly to select an element(s) with a certain identifier. it is symbolized by the # symbol followed by an identifier name. so the HTML element "id" attribute serves as a special indentification number

Step 1: HTML

Give your "p" element the id "my-paragraph"

Step 2: CSS

Navigate to your css file and add a class.
